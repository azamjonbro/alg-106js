cd file -> filega kir
cd .. -> filedan chiq

mkdir -> make directory
nano -> file create
dir -> directory review
pwd -> sizni qaysi fileda yoki papkada turganizzi korsatadi
rm -> remove
rm -rf -> remove file or folder

npm install -> npmdan yuklash

scripts ->bu codelar ishga tushishi uchun yoki qandaydir amal bajarish uchun yoziladigan commands template hisoblanadi.

dependencies-> bu esa o'zida qaysi kutubxona ishlatilayotgani va uni qaysi versiyada ekanini anglatadi.
